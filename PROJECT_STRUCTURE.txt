CRITICAL FILES TO CREATE MANUALLY:
═══════════════════════════════════
1. .env                                   - Copy from .env.example and configure
2. venv/                                  - Create with: python -m venv venv


FILE PERMISSIONS (Linux/macOS):
════════════════════════════════
chmod +x manage.py
chmod 755 app_core/management/commands/setup_system.py


SETUP CHECKLIST:
════════════════
□ Create virtual environment: python -m venv venv
□ Activate virtual environment
□ Install dependencies: pip install -r requirements.txt
□ Copy .env.example to .env
□ Create directories (see above)
□ Run migrations: python manage.py migrate
□ Create superuser: python manage.py createsuperuser
□ Download embedding model: python manage.py setup_system
□ Install Ollama: curl -fsSL https://ollama.ai/install.sh | sh
□ Pull LLM model: ollama pull llama3.2:3b
□ Start Ollama: ollama serve
□ Start Django: python manage.py runserver
□ Access: http://localhost:8000
